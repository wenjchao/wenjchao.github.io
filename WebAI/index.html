說明與需求
  • 模型：Gemma-3n-E2B-it（4.9 GB，品質與中文最好）與 LFM2-VL-450M（0.55 GB，輕量、英文為主，量化組合與 Liquid AI 官方 WebGPU demo 相同）。皆免登入下載。推論引擎：Transformers.js v4。
  • 瀏覽器：建議 Chrome / Edge 最新版。WebGPU 模式需要 GPU；WASM(CPU) 模式 LFM2 可順跑、Gemma-3n 會超過 wasm 4 GB 上限而失敗。
  • 記憶體：Gemma-3n 載入峰值約 7–9 GB、穩定 5–7 GB（建議 16 GB 機器）；LFM2-VL 約 1 GB 內。
  • 下載與選資料夾使用 File System Access API（Chrome / Edge）。Firefox / Safari 只能用「選擇資料夾」備援載入，無法網頁內下載。
  • 所有推論都在本地進行；除了第一次下載模型外，不傳送任何資料到網路。
