# 累積釋放曲線重建

1. 引用自哪篇 paper: hybrid-vesicle-bioink-microrna-delivery
2. Outline (任務主線): 累積釋放曲線重建
3. Method: 
   作者要把「NV 從水膠裡跑出來的速度」量化成可比較的曲線，但 NV 是奈米級脂質囊泡、無色透明、儀器看不到它「跑出來幾顆」。所以先用一種親脂性綠色螢光染料 (PKH67) 鉤進 NV 脂膜雙層，與 NV 形成穩定共軛——只要 NV 跑出水膠，膜上的染料也跟著跑出來，量上清液綠光強度就等於量 NV 釋出量。接著把 100 µg ml⁻¹ PKH67-NV 包進水膠後放進 48-well plate (每孔 0.5 ml PBS、37 °C)，在第 1、2、4、6、8、10、12、14 天分別取 0.5 ml 上清液量螢光。先用一系列已知濃度的 PKH67-NV 在 Ex 490 / Em 502 nm 建立「濃度 → 亮度」校正曲線，每次取樣亮度往線上對，反推 µg ml⁻¹，乘以取樣體積就是「這個時間點到上次之間，釋出多少 µg」。

   這個實驗最容易誤解的地方是「reservoir 每次都被重設」：每個時間點取出 0.5 ml 上清液量螢光後，那 0.5 ml 整管丟掉、補進新 PBS，等於「茶包繼續泡、杯子裡的水換新」。所以下次量到的螢光只反映「上次取樣到這次取樣之間」新釋出的部分，不包含之前已經被倒掉的。要重建「水膠從第 0 天到目前累積總共釋出多少」，必須把每個時間點測到的量逐次相加：Released_t = Released_{t−1} + (concentration_t × 0.5 ml)，直到亮度歸零，再除以「初始包進去的總 NV 量」乘 100，得到 % cumulative release。直接看「當前杯子裡的螢光」會嚴重低估累積——例如第 14 天的數字只代表 12—14 天那段釋出量。

   取樣時間點 1、2、4、6、8、10、12、14 天的不均勻設計是有意的：水膠釋放典型有兩段——初期 (1—2 天) 是「爆釋」(burst release)，表面附近的 NV 一下就跑出來、曲線陡升；之後進入「持續釋放」(sustained release)、曲線變平。如果整 14 天都用均勻間隔取樣，會漏掉前期爆釋細節、看不出三種 NV burst phase 差異；如果整個都加密，後期平緩段會累積大量重複資料浪費樣本。「前期密 (1, 2)、中後期 2 天一次」的不均勻安排剛好把測量精度分配在曲線形狀變化最快的區段。

   這套方法有兩個容易壞掉的地方。一是 PKH67 在 37 °C × 14 天會自然衰減——儀器照射時被光漂白 (photobleaching)、膜上染料分子也會隨時間脫落或被酯酶水解。不校正的話，後期亮度混入了「真實釋出 + 染料自然衰減」兩個訊號，cumulative release 被低估、嚴重時會出現「曲線往下走」的物理不可能反例。實務上要用平行對照組 (同一批 PKH67-NV 不包水膠、直接放 PBS) 跟蹤衰減做校正。二是 burst phase 時上清液濃度可能超出校正曲線線性範圍——螢光分子互相擋光 (self-quenching) 或被周邊染料吸收 (inner-filter effect)，亮度偏離直線甚至下降，反推濃度會被嚴重低估、burst phase 的高度被人為壓低。所以實務上要先用稀釋系列確認樣本落在線性區。
4. 工具與材料: 
   - **PKH67**: 親脂性綠色螢光染料，分子尾巴鉤進 NV 脂膜雙層形成穩定共軛，把不可見的 NV 翻譯成可量化的螢光訊號。
   - **Calibration curve (校正曲線)**: 用一系列已知濃度 PKH67-NV 在 Ex 490 / Em 502 nm 建立「濃度 → 亮度」線性對照，把樣本亮度反推回 µg ml⁻¹。
   - **Cumulative release (累積釋放)**: Released_t = Released_{t−1} + (concentration_t × 0.5 ml) 逐次相加，再除以初始總 NV 量乘 100，避免「reservoir 重設」造成低估。
   - **Burst release**: 釋放曲線初期 (1—2 天) 的陡升段；表面附近的 NV 快速跑出來。取樣間隔在這段要加密。
   - **Sustained release**: 釋放曲線後期的平緩段；NV 緩慢、持續從水膠基質擴散出來，是 Lip / hELs 比 EVs 慢的關鍵。
   - **Photobleaching**: 螢光分子被光長期照射後失去發光能力，會讓後期讀數混入染料衰減訊號。
   - **Inner-filter effect**: 高濃度螢光樣本中，發出的光被周邊染料分子吸收，使亮度偏離與濃度的線性關係，類似 self-quenching。
5. 與此篇文章的關係: 
   在《Hybrid extracellular vesicles-liposome incorporated advanced bioink to deliver microRNA》這篇文章中，作者為了證明 hELs 相對於 EVs 有「沒一次漏光」的緩慢釋放優勢，採用了 PKH67 累積釋放曲線重建。它解決了「每次取樣都把 reservoir 重設導致瞬時讀數無法反映累積」的瓶頸，吃進 14 天 8 個時間點的上清液螢光讀數，產出三種 NV 的累積釋放百分比曲線，與 SEM 孔徑、AFM-DMT 模數三方交叉驗證 NV-GelMA 鍵結強度排序。
