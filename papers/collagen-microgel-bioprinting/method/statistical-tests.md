# 統計檢定與軟體

1. 引用自哪篇 paper: collagen-microgel-bioprinting
2. Outline (任務主線): 統計檢定與軟體
3. Method: 
作者把三類量化結果——孔隙分數、細胞存活率、細胞圓度——分別丟進統計分析軟體 (GraphPad Prism v9.5)，幫每類數字挑一款合適的「裁判」跑假設檢定。每種裁判都會吐出一個 p 值，也就是「假設兩組其實沒差時，光靠運氣看到目前差異或更大差異的機率」；這個機率小於 0.05 就判定差異不是巧合（這是生醫領域長年沿用的慣例門檻）。每組條件至少重複三次 (n ≥ 3)，最後以「平均值加減一個標準差 (mean ± SD)」呈現——平均值是典型落點，標準差告訴你組內散得多開，圖上的誤差棒就是這個 SD。

三類數字各自配對到不同的裁判。孔隙分數同時受兩個變項影響——一個是化學版本 (SPAAC vs. PHYS)，另一個是列印剪切率，所以作者用同時看兩個變項影響的檢定 (two-way ANOVA)：它能拆出「化學版本」「剪切率」與「兩者交互作用」各自貢獻多少差異，避免兩個變項的效果被混在一起。細胞存活率只有一條變項造成組別差異，因此用比較三組以上平均數的檢定 (one-way ANOVA) 就夠了。不論哪一版 ANOVA，它都只能回答「整組裡至少有一對真的不一樣嗎？」，不會告訴你是哪兩組差，所以作者後面再接 Tukey 事後檢定 (Tukey post-hoc)：把所有可能的兩兩組合都比一遍（例如 PHYS-慢 vs. SPAAC-快），並同時把「比很多次容易假陽性」的問題壓住。先看整體有沒有差，再點名是哪對在差，是兩階段的分工。

細胞圓度則不能用 ANOVA。因為 ANOVA 背後假設「每組資料大致呈常態分布」(鐘形曲線)，而細胞圓度天生是高度偏態的——多數細胞縮成圓球時圓度擠在接近 1.0 的牆邊，少數伸出觸手的細胞圓度掉到很低，整組數字長得像一座斜山。作者改用不假設資料呈常態分布的排序型檢定 (Mann–Whitney)：它不去看絕對數字，而是把兩組混在一起重新排序，比較哪一組整體排名比較前面。再加「雙尾 (two-tailed)」是事先不假設哪邊大、只要有差就算數。樣本數則設為 n ≥ 3，因為要算出標準差至少需要 3 個數字。

為什麼挑對裁判這麼重要？若把高度偏態的細胞圓度也硬塞給 ANOVA，平均值與標準差會被那條長尾整個拉歪，可能誇大差異（假陽性）也可能漏掉差異——得出「統計上顯著但生物意義錯誤」的結論。若 ANOVA 後跳過 Tukey、改用「跑很多次 t 檢定」一一比，每比一次都有 5% 假陽性，累積比五次就把假陽性風險推到約 23%，等於自動撈到假的顯著結果。所以挑檢定不只是格式問題，而是直接決定 p 值有沒有意義。
4. 工具與材料: 
   - **GraphPad Prism v9.5**: 生醫常用的統計分析軟體，作者用它對三類數據各自跑對應的假設檢定。
   - **p 值**: 假設「兩組其實沒差」時，光靠運氣看到目前差異或更大差異的機率；p < 0.05 即判定差異非巧合。
   - **顯著水準 p < 0.05**: 生醫領域慣例門檻，低於此值就稱差異為統計顯著。
   - **n ≥ 3**: 每組條件最少要有 3 個重複，因為計算標準差至少需要 3 個數字。
   - **mean ± SD (平均值 ± 標準差)**: 每組數據的呈現方式：平均值是典型落點，標準差是組內散布程度，對應圖上的誤差棒。
   - **one-way ANOVA**: 比較三組以上平均數、變項只有一條時的檢定；本研究用在細胞存活率比較。
   - **two-way ANOVA**: 同時看兩個變項影響的檢定，可拆出兩個變項與兩者交互作用各自的貢獻；本研究用在孔隙分數 (SPAAC vs. PHYS × 不同剪切率) 比較。
   - **Tukey post-hoc**: ANOVA 之後接的事後檢定，能把所有兩兩組合都比一遍並控制假陽性累積；用來點名到底哪兩組在差。
   - **Mann–Whitney 非參數檢定**: 不假設資料呈常態分布、改以排序比較兩組的檢定；用在高度偏態的細胞圓度資料。
   - **two-tailed (雙尾檢定)**: 事先不假設哪一組比較大、只要有差就算數的檢定設定；本研究的 Mann–Whitney 採此版。
5. 與此篇文章的關係: 
   在《Embedded 3D Bioprinting of Collagen Inks into Microgel Baths to Control Hydrogel Microstructure and Cell Spreading》這篇文章中，作者為了證明「孔隙分數隨剪切率變化」「SPAAC 與 PHYS 細胞存活率相當」「大孔組細胞圓度顯著低於對照組」這三項核心結論，採用 GraphPad Prism v9.5 為三類資料各自挑對應檢定 (two-way ANOVA、one-way ANOVA、Mann–Whitney)。這解決了「資料分布不同卻誤用同一檢定」會誇大或漏掉差異的瓶頸，把前面所有量化結果（孔隙、活/死、圓度）轉成讀者能直接看 p 值客觀比較的證據。
