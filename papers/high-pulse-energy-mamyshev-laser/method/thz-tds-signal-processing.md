# THz-TDS 訊號處理鏈

1. 引用自哪篇 paper: high-pulse-energy-mamyshev-laser
2. Outline (任務主線): THz-TDS 訊號處理鏈
3. Method: 
光導天線 (PCA, photoconductive antenna) 是一片半導體「光控開關」：脈衝光照到它瞬間材料變導電，外加的 THz 電場驅動電子穿過形成電流。問題是這個電流只有皮安等級 (10^−12 A)，任何電子放大電路在低頻段都有一種隨頻率反比增加的雜訊叫 1/f 雜訊，直接接示波器訊號會完全被淹沒。作者的辦法是「同步調變 + 鎖相放大」：讓發射端 PCA (Tx) 的偏壓由 Falco WMA-300 高壓放大器配 Keysight 33622A 以 ~23 kHz 方波切換 (0–100 V)，意思是 THz 場每秒被「打開—關閉」23000 次。偵測端 PCA (Rx) 接收到的微弱電流也帶著 23 kHz 指紋。鎖相放大器 (lock-in amplifier，SRS SR630，內建 10^6 V/A 跨阻把電流換成電壓) 把 Rx 訊號乘上 23 kHz 參考訊號再積分——數學上等效於「中心在 23 kHz、頻寬窄到幾 Hz」的極窄帶通濾波器：23 kHz 附近的訊號搬到 DC 留下、其他頻率的雜訊全部擋掉。為什麼選 23 kHz？因為 1/f 雜訊在那麼高的頻率已經衰減到很小。順帶一提，作者刻意不在泵浦光路上加光隔離器：每個隔離器會帶來 ~0.5 dB 插損與額外色散，會把脈衝拉長；為了保持 ~200 fs FWHM 抵達 PCA 才能得到 5 THz 頻寬，作者寧可犧牲反射保護。

整套量測靠 Thorlabs ODL100 自由空間延遲線雙程 20 mm/s 來回掃描建構時域波形，但每次掃描的主峰位置會因機械抖動有幾飛秒到皮秒的隨機偏移 (jitter)。如果幾百次掃描直接疊起來取平均，每次主峰位置不一樣會把平均後的主峰寬度抹糊掉，最尖銳的高頻成分先消失——時域主峰變寬等於頻域高頻成分被衰減，5 THz 那端會直接看不到、頻寬縮到 ~2 THz、90 dB 動態範圍也達不到。作者的做法是先找出每次掃描的主峰位置、在時間軸上對齊到同一點，再做平均；這樣多次掃描的主峰精確疊在一起，訊雜比上升 √N 倍而不損失波形細節。總量測時 ~5 min，由 NI USB-6216 兩通道數位化。

把對齊平均後的時域波形轉成頻域光譜的辦法是加窗快速傅立葉變換 (windowed FFT)：先給時域波形乘上一個平滑的窗函數壓低邊緣不連續再做 FFT，避免邊緣造成「假譜線」。頻率解析度 Δf 由窗長 T 決定 (Δf ≈ 1/T)，窗約 143 ps 就得到 Δf ≈ 7 GHz——剛好夠分辨水氣 10–100 GHz 寬的吸收線。樣品吸收用 A(f) = −ln T(f) 計算，其中 T(f) = P_sample(f)/P_reference(f) 是樣品穿透譜除以空樣品譜。為什麼用自然對數？因為物理上吸收遵守比爾-朗伯定律 T = exp(−αd)，所以 A(f) = α d 直接等於「吸收係數 × 厚度」這個物理量，可以直接拿來查文獻——若用 dB 就要再轉換。最後再用 HITRAN2024 線表 (Gordon et al., JQSRT 353, 109807, 2026) 對照水氣吸收線位置：THz 區段水氣吸收線非常密集，光路上的殘留空氣會在譜上留下幾十條尖銳線，若不扣除可能把 0.557 THz 的 H2O 線誤認為乳糖樣品的特徵，把「找到乳糖 0.53 THz 吸收」這個結論判錯。逐條對照 HITRAN2024 才能正確標出「這條是水氣、那條才是樣品」。

4. 工具與材料: 
   - **光導天線 (PCA, photoconductive antenna)**: 半導體「光控開關」，脈衝光照到瞬間導電，把 THz 電場轉成皮安級電流。
   - **1/f 雜訊**: 電子放大電路在低頻段隨頻率反比增加的雜訊，是 PCA 弱訊號量測的主要敵人。
   - **Lock-in amplifier (SRS SR630)**: 鎖相放大器，內建 10^6 V/A 跨阻；把訊號乘以同頻參考再積分，等效於極窄帶通濾波。
   - **~23 kHz 方波偏壓調變**: Tx PCA 偏壓由 Falco WMA-300 + Keysight 33622A 以 23 kHz 方波切換 0–100 V，給 THz 訊號打上「指紋」。
   - **Thorlabs ODL100 延遲線**: 自由空間延遲線，雙程 20 mm/s 來回掃描建構 THz 時域波形。
   - **Temporal alignment**: 在平均前先找出每次掃描主峰位置、對齊到同一時間點，避免主峰被抹糊損失高頻成分。
   - **Windowed FFT**: 對時域波形乘上窗函數再做 FFT，避免邊緣不連續造成假譜線；窗長 ~143 ps 給出 ~7 GHz 解析度。
   - **吸收譜 A(f) = −ln T(f)**: 用自然對數計算吸收，使 A(f) = α d 直接對應 Beer-Lambert 物理量。
   - **HITRAN2024 線表**: 高解析度大氣分子吸收線資料庫 (Gordon et al., JQSRT 353, 109807, 2026)，用來識別並扣除 THz 譜上的水氣干擾。
   - **NI USB-6216 DAQ**: 兩通道數據擷取卡，把 lock-in 輸出與延遲線位置同步數位化。
5. 與此篇文章的關係: 
在《High-pulse-energy integrated mode-locked laser using a Mamyshev oscillator》這篇文章中，作者要證明這顆晶片 Mamyshev oscillator 能驅動一整套有用的 THz 時域光譜儀，採用 THz-TDS 訊號處理鏈。它解決的具體瓶頸是「PCA 輸出皮安級電流被 1/f 雜訊與水氣干擾蓋過、無法看出樣品特徵」這件事；它吃 Sec. 2-J 的 PCA 弱電流訊號與 Thorlabs ODL100 延遲線時域波形進來，產出動態範圍 90 dB、頻寬 5 THz 的乾淨光譜，餵給後續的矽晶圓厚度量測 (523 µm) 與乳糖/麵粉材料識別 (0.53 THz 特徵) 結論。
