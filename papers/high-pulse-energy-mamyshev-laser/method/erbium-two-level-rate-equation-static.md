# Erbium 增益的雙能階速率方程靜態模型

1. 引用自哪篇 paper: high-pulse-energy-mamyshev-laser
2. Outline (任務主線): Erbium 增益的雙能階速率方程靜態模型
3. Method: 
在 0.42 公尺的 Er:Si3N4 雷射腔裡，作者放棄文獻常用的捷徑「固定飽和功率」(saturation power, P_sat) 公式 g = g_0 / (1 + P/P_sat)。這條公式把整條波導當作均勻、泵浦取之不竭的盒子，在幾公尺長、低摻雜的傳統 erbium 光纖放大器裡尚可用；但本工作的波導只有半公尺長、erbium 峰值濃度推到 3.68 × 10^26 m^-3 (≈ 0.4 at.%)，1480 nm 泵浦光進入波導幾公分內就被吸掉一半，沿位置完全不均勻。硬套 P_sat 會把整條增益曲線拉高、和實驗對不上。作者改成直接列出上能階粒子數 N_2 與下能階粒子數 N_1 的時間平衡式 (Eq. 2)，假設達到穩態 (dN/dt = 0) 後解出每個 z 位置的反轉粒子數比例 n(z) = N_2/N_0'（Eq. 3），再代回沿位置變化的增益 g(z) 與泵浦衰減 dI_p/dz（Eq. 4）。

Eq. 2 的 dN_2/dt 本質是上能階離子的「收支表」：第一項 −N_2/τ 是自己掉下來 (spontaneous emission)，要等 τ ≈ 3 毫秒；第二項 (N_1 σ_12s − N_2 σ_21s) φ_s 是訊號光把下能階吸收上去 (σ_12s)、再把上能階踢下來放出新光子 (σ_21s)；第三項對 1480 nm 泵浦同理，但 σ_12p 遠大於 σ_21p，淨效果是把離子推上去。穩態假設 (dN/dt = 0) 站得住腳的原因，是雷射重複率 1.755 億 Hz、脈衝間距 5.7 ns，遠快於 3 ms 的離子壽命；離子根本跟不上單發脈衝，只看得到平均功率。這個近似把「µs 級族群動力學」與「ns 級脈衝動力學」乾淨分開，避開直接模擬要在 ns 步長下追蹤 ms 時間的巨量計算。

把摻雜推到 0.4 at.% 還會多出一個麻煩：erbium 離子會兩兩三三擠成「聚集 (cluster)」，cluster 裡的離子照吸光卻把能量在彼此間以非輻射方式消耗掉、不放新光子。作者用比例 n_cluster ≈ 6% 描述這群「死掉的離子」，在增益式 (Eq. 4) 中只讓 (1 − n_cluster) 那部分參與放射、卻讓所有離子都參與吸收 (修正參考 Delevaque 1993 的 pair-induced quenching)。吸收與放射截面 σ_12 / σ_21 不能照抄文獻 —— 不同基質差異大 —— 而是用同一片晶圓做的兩組獨立量測逆推：用 erbium 螢光衰減曲線 (photoluminescence decay) 抓壽命 τ 與 cluster 比例，再用短波導放大器量小訊號增益對泵浦功率的響應來定 σ。

解完後得到的讀數是「每個 z 位置的本地增益 g(z)」這條沿波導的曲線，直接餵進 GNLSE 的增益項 (g − α) A / 2；同時也得到泵浦光衰減曲線 I_p,prop(z)，驗證雙端泵浦下波導兩端較強、中段被吸到較弱的非均勻分佈。Fig. S8 顯示腔內脈衝能量在兩端往上爬、中間下沉一塊，正是泵浦從兩端注入、沿途被吸光、中段反轉粒子數不足的指紋 —— 而這正是固定 P_sat 模型做不出來的 U 型曲線。如果偷懶套 P_sat 或把 n_cluster 設為 0，預測的小訊號增益會偏高好幾 dB，鎖模門檻被拉低，後續 Algorithm 1 畫出來的穩定區也跟著錯位。

4. 工具與材料: 
   - **兩能階速率方程 (two-level rate equations)**: Eq. 2，描述上下能階粒子數隨自發放射、訊號與泵浦誘導吸收放射隨時間變化的方程組。
   - **反轉粒子數比例 (population inversion, n = N_2/N_0')**: 上能階離子佔活性離子總數的比例；穩態下由 Eq. 3 直接解出，是本地增益 g(z) 的核心輸入。
   - **吸收/放射截面 (σ_12, σ_21)**: 把光子通量轉成躍遷率的比例係數；本工作量測校準到 σ_12p = 3.43 × 10^-25、σ_12s = 2.53 × 10^-25、σ_21p = 5.9 × 10^-26、σ_21s = 4.48 × 10^-25 m^2。
   - **自發放射壽命 (spontaneous emission lifetime, τ)**: 上能階離子平均待 3 ms 後自發掉回下能階；遠長於 5.7 ns 的脈衝間距，這是穩態近似成立的關鍵。
   - **聚集無效離子比例 (n_cluster)**: 0.4 at.% 高摻雜下約 6% 的 erbium 離子擠成 cluster，吸收訊號卻不放新光子，需在增益式中扣除。
   - **模場-摻雜重疊因子 (overlap factor, Γ)**: 光模強度分佈與 erbium 摻雜剖面的歸一化重疊；本工作 Γ ≈ 0.22。
   - **螢光衰減量測 (photoluminescence decay)**: 獨立量測同片晶圓短波導的螢光隨時間衰減，用來逆推 τ 與 cluster 比例。
   - **高折射率對比修正 (n_g/n_A)**: Robinson 2008 提出，用群速度折射率 n_g 與材料折射率 n_A 比值修正高侷限波導中的增益。
5. 與此篇文章的關係: 
在《High-pulse-energy integrated mode-locked laser using a Mamyshev oscillator》這篇文章中，作者要在 0.42 公尺的 Er:Si3N4 波導上做出穩態鎖模解，遇到的瓶頸是傳統「固定 P_sat 飽和模型」會把高摻雜短波導的增益估錯。作者改用兩能階速率方程靜態模型，吃進泵浦/訊號強度與校準過的 σ、τ、n_cluster，直接解出沿位置變化的增益 g(z)。這個 g(z) 就是下游 Algorithm 1 自洽穩態搜索演算法做 GNLSE 模擬的核心增益輸入。
