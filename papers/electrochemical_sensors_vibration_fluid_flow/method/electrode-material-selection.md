# 電極材料與電化學量測架構選型

## 主線
為不同分析物 (重金屬、有機分子、生物標誌物) 選擇對應的工作電極材料、參考/輔助電極與伏安/電流量測技術，作為後續導入振動或流動的基板平台。

## 技術解析
整篇 review 把電化學感測器當成一間「驗貨櫃台」，三根電極各有分工：工作電極 (WE) 真的在跑反應、參考電極提供穩定的電位零點、輔助電極 (counter) 把電流拉走。如果只用兩根，「跑反應」和「當參考」會擠在同一根，電流一大就把電位零點拖歪。整套系統的兩條控制方程是 Nernst 方程 $E = E^{0} + \frac{0.0592}{n}\log[\mathrm{Ox}]$ (Eq. 2，電位 ↔ 濃度) 與 Faraday 定律 $N = Q/(nF)$ (Eq. 3，電荷 ↔ 反應分子數)：前者是靜態天平、撐起電位感測器；後者是動態流量計、撐起電流與伏安感測器。選材的邏輯是「拿手電位窗 + 對目標分子的親和力」：金 (Au) 親汞齊化、是 Pb/Cu/Cd 剝離分析的預設材料；鉑 (Pt) 反應快、適合 Mn；玻璃碳電極 (glassy carbon electrode, GCE) 與篩網印刷電極 (screen-printed electrode, SPE) 是便宜、好修飾的裸基材，能再鍍奈米線或抗體當作生物感測平台。

材料光譜裡還有第四位特別重要的成員。硼摻雜鑽石電極 (boron-doped diamond, BDD) 之所以對有機分子特別好用，關鍵在「電位窗」：每種電極材料都有一段在水裡能用的電壓範圍，太正會把水氧化、太負會還原成氫，這些副反應電流會把目標訊號淹掉。BDD 的鑽石骨架特別惰性，水分解門檻被推到很遠、背景電流也低一個數量級——河水裡的 Cd 剝離峰、epinephrine、acetaminophen 這些落在金鉑會被淹沒位置的訊號，在 BDD 上才看得出來。

選好材料還要選對「拷問節奏」——五把量測技術其實是「同一個氧化還原反應用不同節奏拷問一遍」。陽極溶出伏安法 (anodic stripping voltammetry, ASV) 先把金屬電鍍到電極上累積幾分鐘、再反掃一次氧化釋放——這一步累積就是天然放大器，是 Table 4 多數重金屬 LOD 能壓到 nM 甚至 pM 的功臣。微分脈衝伏安法 (differential pulse voltammetry, DPV) 與方波伏安法 (square wave voltammetry, SWV) 在掃描中夾入小脈衝、只取脈衝前後的電流差，把和電壓無關的充電背景扣掉，訊雜比高出一截，是有機分子與藥物的主力。定電位電流法 (amperometry) 把電位釘住、一直讀穩態電流，適合連續監測流動樣品。阻抗譜 (electrochemical impedance spectroscopy, EIS) 灌小 AC 訊號掃頻率、用 Nyquist 圖讀「界面有沒有被抗體–抗原貼上一層」，是生物標誌物的主力。技術錯配比材料選錯更致命：拿 amperometry 量 pM 級 Pb，訊號直接落在背景之下，後續再加振動或流動都救不回來。

落到工程實作面，後續振動 / 流動實驗多半選 SPE 而非實驗室那種大 GCE，因為 SPE 把三根電極一次印在塑膠片上、規格固定、用完即拋——振動實驗常要把馬達整個跟電極黏在一起 (Wang et al. 2019、Liu et al. 2021)，SPE 一次性能避免修飾被破壞；流動實驗則需要平面外形塞進 PDMS 通道。整套系統還有一條隱形紅線：電流只走 WE↔counter、參考電極絕對不能有電流，否則它的電位會跟著電流漂、剝離峰每次都跑位，後面再怎麼壓擴散層也定不出量。

## 工具/材料/方法清單
- **三電極系統 (WE / counter / reference)**：工作電極跑反應、參考電極提供電位零點、輔助電極把電流拉走的標準架構。
- **Nernst 方程 (Eq. 2)**：$E = E^{0} + \frac{0.0592}{n}\log[\mathrm{Ox}]$，描述不通電流時電位與濃度的對數關係，是電位感測器的控制方程。
- **Faraday 定律 (Eq. 3)**：$N = Q/(nF)$，每跑一個分子要 n 顆電子，是電流與伏安感測器把電量回推分子數的會計簿。
- **Au 工作電極**：親汞齊化、能把 Pb/Cu/Cd 電鍍上來再剝下，是重金屬 ASV 的預設材料。
- **Pt 工作電極**：反應快、電位窗較窄，適合 Mn 等氧化態多變的金屬。
- **玻璃碳電極 (GCE)**：便宜、好修飾的裸基材，適合再鍍奈米結構與抗體做生物感測。
- **硼摻雜鑽石電極 (BDD)**：電位窗寬、背景電流低，能撈出落在金鉑水分解區的有機訊號。
- **篩網印刷電極 (SPE)**：三電極一次印在塑膠片上、規格固定、用完即拋，方便耦合振動與微流道。
- **陽極溶出伏安法 (ASV)**：先沉積累積幾分鐘、再反掃一次釋放，是重金屬痕量分析的天然放大器。
- **微分脈衝伏安法 (DPV) / 方波伏安法 (SWV)**：掃描中夾脈衝、只取電流差，扣掉充電背景以提升訊雜比。
- **定電位電流法 (amperometry)**：把電位釘住一直讀穩態電流，適合連續監測流動樣品。
- **阻抗譜 (EIS)**：灌小 AC 訊號掃頻率、量阻抗實虛部，主要用來讀生物標誌物的界面變化。
