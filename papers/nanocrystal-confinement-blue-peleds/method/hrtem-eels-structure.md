# 高解析 TEM、HRTEM 與電子能量損失譜 (EELS) 結構表徵

1. 引用自哪篇 paper: nanocrystal-confinement-blue-peleds
2. Outline (任務主線): 高解析 TEM、HRTEM 與電子能量損失譜 (EELS) 結構表徵
3. Method:
   TEM 樣品必須夠薄（< 100 nm）讓電子能穿透，又要保留薄膜真實的結晶狀態。作者用兩種製樣方式互相印證：第一種把鈣鈦礦前驅液直接旋塗在碳膜銅網（一面附碳膜的金屬網片，可當 TEM 樣品台）上、按薄膜製程退火；第二種先在真正的 ITO/傳輸層基板上旋塗、退火完成，再用甲苯把鈣鈦礦刮下重新分散到銅網。HRTEM 用像差校正 TEM (Cs-corrected TEM) JEOL Grand ARM 300 (300 kV)——這台機器配備能修正電子透鏡固有像差的元件，把解析度推到 0.1–0.2 nm，能直接看到單一原子柱排成的點陣。判讀晶相靠『快速傅立葉變換衍射圖 (FFT)』：把點陣影像丟進 FFT，原本連續的點陣會變成一組亮點，亮點間距反映晶面間距、亮點對稱性反映晶體對稱類型。觀察角度叫『帶軸 (zone axis, ZA)』，用三個整數標示（如 [223]）。粒徑統計用 Nano Measurer 軟體：每顆奈米晶量邊長、每樣本 ≥ 100 顆、分散在多區域多樣本上採樣、排除重疊顆粒，最後以平均值 ± 標準差報告。

   鈣鈦礦是離子晶體，對電子劑量非常敏感——一般 TEM 強束流會把鹵離子撞鬆、表面非晶化，FFT 看到的會是『非晶玻璃』而不是立方相，作者的相變宣稱就會被質疑成『電子束燒出來的假象』。EELS 也會被驅趕的鹵離子擾亂、無法分辨表面包覆。所以作者特別用『Gatan K2 ultra-low-dose 相機』——這台相機把每個電子訊號數位化，極微弱的訊號就能成像；因此電子劑量比常規拍低一個數量級，樣品幾乎不被破壞，HRTEM 看到的真的是原始晶相。低劑量 + Gatan K2 不只是『更精細』，是『讓相變主張站得住腳的必要條件』。

   怎麼從 HRTEM 影像判斷立方相還是正交相？正交相（晶格軸長不等）的 FFT 斑點分佈不對稱、立方相（三軸等長且互相垂直）則呈現正方形對稱。但單一帶軸不夠：從特定角度看立方相與某些四方相 (tetragonal) 的 FFT 會幾乎一樣，所以作者沿多個帶軸拍 FFT。正交相的 pristine 樣品沿 [223] 與 [332] 兩個帶軸都拍到正交相特有的低對稱斑點；OEGA 樣品則沿多個帶軸都拍到高對稱的正方形斑點，才能排除四方相、確認真的是立方相。

   光看相變還不夠——作者還要直接證明 OEGA 真的形成『表面薄包覆層』。電子穿過樣品時會把部分能量交給樣品原子的內層電子，每種元素有自己的『門檻能量』（O-K edge ~532 eV、Cs-L edge ~726 eV）。作者把高角環狀暗場掃描 TEM (HAADF-STEM) 的電子束沿奈米晶剖一條線、逐點記錄能量損失，就是電子能量損失譜 (EELS)。挑兩個關鍵元素互補：O 是 OEGA 獨有（鈣鈦礦只有 Cs/Pb/Br），所以 O 訊號出現的地方就是 OEGA 所在；Cs 是鈣鈦礦的 A 位陽離子、標誌晶粒本體。把兩條剖線疊在一起，氧訊號剛好分佈在 Cs 訊號的兩側（即奈米晶邊緣），就直接證明 OEGA 包覆在奈米晶表面，而非進入體相。

   作者用兩個額外設計確保結論可信。第一，TEM 製樣做兩組：一組直接旋塗在碳膜銅網、一組從真實 ITO/傳輸層刮下重分散。兩組都看到 11 nm 立方相，就排除『碳膜表面化學特殊性影響結晶』的質疑，確認真實元件中的奈米晶就是這個尺寸與相。第二，HRTEM 必須用像差校正 TEM——電子透鏡有固有的球面像差，邊緣電子聚焦點跟中心不同，導致原子點陣被『暈成一團』。Cs-corrector 由多重磁多極元件主動產生反向像差抵消、相當於把眼鏡度數校正，這樣立方相 vs 正交相的對稱差才能量得準。

4. 工具與材料:
   - **像差校正 TEM (Cs-corrected TEM)**: JEOL Grand ARM 300 (300 kV)，配備 Cs-corrector 抵消電子透鏡球面像差，解析度推到 0.1–0.2 nm。
   - **Gatan K2 ultra-low-dose 相機**: 把每個電子訊號數位化的像素偵測相機，可在極低電子劑量下成像，避免鈣鈦礦表面非晶化。
   - **快速傅立葉變換衍射圖 (FFT)**: 把 HRTEM 點陣影像轉成衍射斑點，斑點對稱性對應晶體對稱類型，用以區分立方相與正交相。
   - **帶軸 (zone axis, ZA)**: 從晶體某方向觀察的軸向，用三個整數標示（如 [223]）；作者沿多個帶軸採 FFT 避免單一視角誤判。
   - **HAADF-STEM (High-Angle Annular Dark-Field STEM)**: 暗場掃描成像，讓重元素位置高度反差，作為 EELS 剖線的成像基礎。
   - **電子能量損失譜 (EELS)**: 逐點記錄電子穿樣品損失的能量，O-K 邊 (~532 eV) 與 Cs-L 邊 (~726 eV) 各標誌 OEGA 與鈣鈦礦本體。
   - **正交相 (orthorhombic, Pnma)**: 晶格軸長不等的低對稱相，pristine 鈣鈦礦的常溫相。
   - **立方相 (cubic, Pm-3m)**: 三軸等長且互相垂直的高對稱相，OEGA 處理後的常溫相。
   - **Nano Measurer 軟體**: 量取奈米晶邊長的圖像分析工具，每樣本 ≥ 100 顆、報告 mean ± SD。

5. 與此篇文章的關係:
   在《In situ nanocrystal confinement for efficient blue perovskite LEDs》這篇文章中，作者主張 OEGA 把鈣鈦礦從正交相轉成立方相、並在表面形成薄包覆層，這兩個結構主張需要直接的原子級證據才能站得住腳。像差校正 HRTEM + 多帶軸 FFT 解決『分得清立方相與四方相』的瓶頸，EELS 元素剖面解決『證明氧只在奈米晶邊緣』的需求；這些表徵把上游 OEGA 處理後的薄膜直接連到下游的光物理機制解釋。
