# 3D-FDTD 光學耦合模擬：解析次微米粒子對波導模式的破壞

1. 引用自哪篇 paper: nanocrystal-confinement-blue-peleds
2. Outline (任務主線): 3D-FDTD 光學耦合模擬：解析次微米粒子對波導模式的破壞
3. Method:
   作者在電腦裡把整顆 PeLED 拆成一格一格的小立方體，每格標好它是哪一層材料，再用三維時域有限差分模擬 (3D-FDTD) 解 Maxwell 方程式，等於讓光在這個多層三明治裡一步一步『跳格子』地傳出去，最後算出有多少光逃出表面、多少被金屬電極吃掉。模擬的輸入只有兩件事：每一層的厚度與折射率。作者把實際元件依序拆解：玻璃 (n = 1.5) / ITO (n = 1.8, 110 nm) / m-PEDOT:PSS (n = 1.63, 15 nm) / 鈣鈦礦發光層 (n = 2.10 對照 / 2.01 OEGA, 20 nm) / TPBi (n = 1.7, 40 nm) / 鋁電極 (100 nm)；其中鈣鈦礦與 m-PEDOT:PSS 的折射率 n 與吸收係數 k 由橢偏儀 (ellipsometer) 量得，其他層引用 Li 等人 (Adv. Mater. 2021, ref 13)。作者刻意跑兩套模擬，一套是完全平整的三明治、一套是頂層加上實驗 SEM/AFM 量到的次微米顆粒花紋；兩套相減就乾淨切出『顆粒貢獻了多少額外光取出』。

   為什麼次微米顆粒能幫忙撈光？因為鈣鈦礦折射率約 2.0、玻璃只有 1.5，平整介面之下，光只要射得比臨界角 ($\sin\theta_c = n_2/n_1 \approx 0.75$) 更斜就會 100% 被反射回鈣鈦礦裡，沿著薄膜橫著跑，這部分叫『側向波導模式 (lateral-propagating waveguide modes)』，最終會被鋁電極吃光、變成熱損耗。SEM/AFM 看到 OEGA/PEA 薄膜表面均勻撒著一顆顆直徑幾百奈米的小丘 (isolated submicrometric particles)，能量散布光譜 (EDS) 顯示它們的 Cs:Pb:Br 比例與本體相同，作者因此推測是 11 nm 奈米晶在表面又聚集而成的鈣鈦礦小丘。這些小丘打破平整介面、把原本只能側向跑的光散射成各種方向，部分撞到小丘的角度因此能翻過臨界角、逃進玻璃並射向觀眾。FDTD 把『加顆粒 vs. 不加顆粒』兩種情境的逃出比例直接算出來 (Supplementary Fig. 30e)，量化這個光取出 (light outcoupling) 的貢獻。

   要算逃出比例，第一件事是處理『發光源的方向』。鈣鈦礦裡的電子-電洞復合可看成一支發光的小天線（偶極），同一支偶極朝不同方向放出的光被多層三明治攔截的比例差很多：橫躺的偶極（x、y 方向，平行於膜面）放出的光容易直接穿出表面，立著的偶極（z 方向）放出的光大半被困住。公式 (3) $P=\Theta(P_x^s+P_y^s)+(1-\Theta)P_z^s$ 因此把總輻射功率拆成『橫躺貢獻』與『站著貢獻』，再用權重 Θ 加權合成。Θ 是橫躺偶極佔的比例，叫『水平偶極比 (horizontal dipole ratio)』；鈣鈦礦的發光偶極方向隨機（各向同性發射體, isotropic emitter），三維裡有兩維是橫的，所以 Θ = 2/3。實作上，FDTD 一次只能放一支單一方向偶極跑模擬，作者因此分別在 x、y、z 各跑一次（多偏振模擬, multiple polarization simulations），再用公式 (3) 加權合成，把『隨機方向發光』翻成可計算的非相干各向同性光源 (參考 Cui et al., Sci. Adv. 2021, ref 71)。

   FDTD 雖然原理清楚，但結果好不好用，靠的是輸入是否誠實，作者在四個地方刻意把關。第一，顆粒形貌必須是真實的：作者把 SEM/AFM 量到的 OEGA/PEA 薄膜表面細節直接導入模型，不是隨便假設一個規則花紋，這樣算出來的才是『這顆元件實際多撈到多少光』，而不是『假設花紋下的理論上限』。第二，折射率必須對齊到實際薄膜：pristine 與 OEGA 兩款各跑一次橢偏儀，獨立量到 n = 2.10 與 2.01 並分別餵進 FDTD；如果隨便共用同一組 n，就會高估被困光比例、把『加顆粒能撈出的光』算得比實際大。第三，偶極比 Θ = 2/3 不能拿掉：把 Θ 設成 1 等於假設所有偶極都橫躺，會吹高取出效率；設成 0 又低估。第四，必須跑『平面』對照：EQE 從 10.7% 跳到 21.8% 同時有兩個來源——奈米晶 PLQY 提升到 83%，以及表面顆粒撈光；只算加顆粒版本就分不清這兩部分各佔多少，靠『平面 vs. 加顆粒』兩套相減，作者才能乾淨把這個 EQE 增益歸給次微米顆粒形貌。

4. 工具與材料:
   - **3D Finite-Difference Time-Domain (3D-FDTD)**: 三維時域有限差分模擬：把元件拆成一格一格小立方體，每隔一小段時間更新每格的電磁場，逐步算光在多層結構裡如何傳遞與逃逸。
   - **Ellipsometer**: 橢偏儀：量測薄膜折射率實部 n 與虛部 k 的標準工具，作者用它取得 m-PEDOT:PSS 與鈣鈦礦層的光學常數（pristine n = 2.10、OEGA 處理 n = 2.01）。
   - **Lateral-propagating waveguide modes**: 側向波導模式：光被夾在高折射率鈣鈦礦薄膜裡橫著跑、出不來的部分，最終被鋁電極吸收成熱損耗。
   - **Light outcoupling**: 光取出：把原本被困在元件裡的光導出表面變成可見發光的物理過程，是 EQE 的乘法因子之一。
   - **Isolated submicrometric particles**: 次微米顆粒：OEGA/PEA 薄膜表面均勻撒佈、直徑幾百奈米的小丘，EDS 顯示為鈣鈦礦組成，推測來自 11 nm 奈米晶在表面再聚集。
   - **Horizontal dipole ratio (Θ)**: 水平偶極比：橫躺（x、y 方向）偶極佔總發光偶極的比例。各向同性發射體 Θ = 2/3。
   - **Isotropic emitter**: 各向同性發射體：發光偶極方向隨機、各方向機率相同的光源；鈣鈦礦屬於這類，因此用 Θ = 2/3 處理。
   - **Multiple polarization simulations**: 多偏振模擬：分別在 x、y、z 方向各跑一次單一方向偶極，再用公式 (3) 加權合成非相干各向同性光源。
   - **Equation (3)**: $P=\Theta(P_x^s+P_y^s)+(1-\Theta)P_z^s$：把三個方向偶極的輻射功率依水平偶極比 Θ 加權成總輻射功率，方法承襲 Cui et al. Sci. Adv. 2021 (ref 71)。

5. 與此篇文章的關係:
   在《In situ nanocrystal confinement for efficient blue perovskite LEDs》這篇文章中，作者為了把藍光 PeLED 的外部量子效率推高到 21.8%，採用了三維時域有限差分模擬 (3D-FDTD)。這個模擬吃進實驗 SEM/AFM 量到的 OEGA/PEA 薄膜表面形貌與橢偏儀量到的折射率，產出『平面 vs. 加次微米顆粒』兩種結構下的光取出效率，解決『EQE 增益裡有多少來自表面顆粒形貌』無法用實驗直接拆分的瓶頸。模擬結果為奈米晶聚集形成的次微米顆粒能抽取側向波導模式提供獨立的光學佐證，讓作者把奈米形貌、光取出、EQE 三者連成完整因果鏈。
