# 組織化學染色 (Trichrome / Von Kossa / 免疫螢光)

1. 引用自哪篇 paper: pediatric-tri-tube-valved-conduit
2. Outline (任務主線): 視覺化 explanted valve 內 collagen 結構、鈣化沉積、細胞遷移與內皮形成模式，將生長與 remodeling 的型態學證據與功能結果連結。
3. Method:

瓣膜從羊體內取出後沒上色之前根本看不出膠原蛋白、鈣化、細胞躲在哪，所以作者先靠染色把不同結構「上不同顏色」再讀。流程從化學「定型」組織開始：tri-tube 樣本用冷 4% 多聚甲醛 (paraformaldehyde) 把蛋白質在離氨酸殘基之間以共價橋鎖住，整個分子網路被瞬間定格，不會被自體酵素分解掉。固定完依後續要做的染色挑包埋方式：tri-tube 要做免疫螢光看細胞身分，就用冷凍包埋——先用 30% 蔗糖把組織泡到飽（避免結冰形成冰晶撕破細胞），再用 OCT 包埋液包起來丟進液氮急凍，低溫過程中蛋白質的抗原決定位 (epitope) 不會被破壞，後續抗體還認得；Contegra 樣本只用 Trichrome 與 Von Kossa 看形態，不需要保留抗原，就用 10% 福馬林 + 石蠟包埋的標準路線，這種高溫滲蠟的做法形態學影像更漂亮、邊界更清晰。最後用切片機切出 9 微米的薄片，貼到玻片上等染。

形態學染色用兩種：Trichrome 看膠原蛋白骨架、Von Kossa 看鈣化點。Trichrome 字面意思是「三色」，用三種染料分別貼上三種不同分子：一種偏酸性染料只貼結構鬆散、間隙大的膠原蛋白纖維，把它染成藍綠色；另一種染料貼比較緻密的非膠原蛋白（如肌肉、細胞質），把它染成紅色；第三種叫鐵蘇木素 (iron hematoxylin) 專門結合細胞核裡帶負電的 DNA，把細胞核染成黑色。一張切片就同時顯示「骨架（藍綠）vs. 細胞質（紅）vs. 細胞核（黑）」三層訊息，作者能直接看膠原蛋白排列是不是整齊、有沒有細胞住進來。Von Kossa 則是把切片泡在硝酸銀溶液裡：鈣化在組織裡是以磷酸鈣 (calcium phosphate) 形式沉積，銀離子會擠掉沉積物裡的鈣、自己跑進去結合磷酸；接著用光照把銀離子還原成金屬銀，金屬銀是黑色的，於是原本看不見的鈣化點就變成清晰的黑色斑點。要注意 Von Kossa 嚴格說是染磷酸鹽不是染鈣，緩衝液若清洗不乾淨可能出現假鈣化黑點，所以這篇也用後續的化學定量 (Ca²⁺ assay) 複驗。

看細胞身分要用免疫螢光，核心是「用一隻會專門認某個蛋白的抗體當探針」。作者先用一抗 (primary antibody) 去貼想看的蛋白：VWF 抗體 (ab6994, 1:200) 認血管內皮特有的 von Willebrand factor、α-SMA 抗體 (A-5338, 1:200) 認平滑肌與肌纖維母細胞的收縮蛋白、Ki67 抗體 (ab15580, 1:100) 認只在分裂中細胞核才會出現的 Ki67 蛋白。一抗本身不發光，所以再加一隻會認一抗物種的二抗 (secondary antibody)，二抗身上掛了 Alexa Fluor 647 螢光分子；每隻一抗周圍能聚集多隻二抗，訊號被放大、暗訊號也能被看到。配合 Hoechst 33342 染所有細胞核，影像就同時呈現「位置 + 細胞身分」兩層訊息。三個 marker 一起回答「住進瓣膜的是哪些細胞、在做什麼」：VWF 看瓣膜表面有沒有長出抗血栓的血管內皮層、α-SMA 看組織管深處有沒有負責支撐與生產膠原蛋白的間質細胞、Ki67 配合時間點判斷宿主細胞是不是還在主動長進來、是不是還在重塑組織。

這些染色有兩個讀圖端的陷阱必須警惕。第一是免疫螢光的假陽性：膠原蛋白和彈性蛋白在 488 nm 附近會自己發出微弱螢光 (autofluorescence)，二抗也可能非特異黏在組織某些區域。如果沒設「同物種非特異 IgG 一抗」或「只加二抗」的陰性對照，無法區分「真正的 VWF 陽性訊號」和「組織自己亮 + 二抗亂黏」，可能高估了內皮覆蓋率、誤判細胞增殖比例。第二是 Von Kossa 嚴格說是染磷酸鹽不是染鈣，緩衝液清洗不乾淨可能造成假鈣化黑點；所以這篇對微鈣化小點保留懷疑空間，並用化學定量（Ca²⁺ assay）複驗大區塊鈣化的存在。整套染色最後放上 Leica DMi8 widefield 螢光顯微鏡拍照，作為 explanted valve 切片的視覺證據。

4. 工具與材料:

   - **paraformaldehyde fixation**: 多聚甲醛固定。在蛋白質離氨酸殘基間建立共價橋，把分子網路定格，避免自體酵素分解組織。
   - **OCT + 液氮 冷凍包埋**: 蔗糖滲透防冰晶 + OCT 包埋液 + 液氮急凍。保留抗原決定位給免疫螢光用。
   - **paraffin embedding**: 石蠟包埋。脫水、滲蠟、加熱固化，做出邊界清晰的形態學切片，但高溫會破壞抗原。
   - **Lillie's Trichrome**: 三色染色。膠原蛋白藍綠、非膠原蛋白紅、細胞核黑，三層訊息同時呈現。
   - **Von Kossa**: 鈣化染色。銀離子置換磷酸鈣沉積中的鈣，再還原成黑色金屬銀。嚴格說是染磷酸鹽。
   - **immunofluorescence**: 免疫螢光。用會專門認某蛋白的抗體當探針，配合螢光二抗讓目標蛋白被照亮。
   - **primary + secondary antibody**: 一抗 + 二抗策略。二抗掛螢光、認一抗物種，達到訊號放大與多重染色。
   - **VWF (von Willebrand factor)**: 血管內皮特有的蛋白標記。看到 VWF 陽性代表瓣膜表面長出抗血栓的內皮層。
   - **α-SMA (alpha smooth muscle actin)**: 平滑肌與肌纖維母細胞的收縮蛋白標記。看到它代表深處有支撐與生產膠原蛋白的間質細胞。
   - **Ki67**: 只在分裂中細胞核才出現的蛋白。標記「正在增殖」的細胞，配合時間點判斷宿主細胞是否還在主動長進來。
   - **Hoechst 33342**: 細胞核染料。結合 DNA，把所有細胞核染成藍色，作為位置參考。
   - **Leica DMi8 widefield**: 本實驗使用的廣域螢光顯微鏡，搭配 DFC 7000T 相機拍攝切片影像。

5. 與此篇文章的關係:

這篇 paper 要回答兒科生物工程肺動脈瓣 (tri-tube valve) 在生長中羊體內 52 週後到底是「真的跟著小孩一起長大、由宿主細胞接管」還是「只是被動撐大、開始鈣化失效」，而光靠超音波看尺寸、力學測 modulus 都無法直接看見瓣膜內部組織層級到底發生什麼事，因此需要組織化學染色把取出後的瓣膜切片、用不同顏色標出膠原蛋白骨架、鈣化沉積、宿主細胞身分與分裂狀態。用 Lillie's Trichrome 與 Von Kossa 可以一次回答「膠原骨架是否完整、有沒有像化學固定生物瓣那樣鈣化失效」這個與 Hancock 豬瓣、Contegra 牛瓣對比的核心訴求，免疫螢光 (VWF、α-SMA、Ki67) 則回答「瓣膜表面有沒有長出抗血栓內皮、深層有沒有間質細胞住進來、它們是不是還在主動繁殖」這個生長機制的關鍵問題。它在整套方法裡扮演結構層面的視覺證據，向下與生化定量 (collagen、DNA、Ca²⁺ assay) 互相校驗、把眼睛印象轉成可量化的密度，向上與超音波、力學測試對接，把「functional surrogate (尺寸與血流)」與「結構 surrogate (細胞與膠原)」綁在一起，才能合理宣稱觀察到的是 somatic growth 而非 aneurysmal stretching；同時為了避免抗原被高溫石蠟破壞，tri-tube 樣本走冷凍包埋路線、Contegra 樣本走石蠟路線，這種包埋分流也是為了讓免疫螢光與形態學染色都能各取所需地服務這篇 paper 的比較設計。
