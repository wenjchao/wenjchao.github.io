# Ki67 增殖細胞定量

1. 引用自哪篇 paper: pediatric-tri-tube-valved-conduit
2. Outline (任務主線): 量化 explanted valve 不同時間點 root 與 leaflet 內處於分裂期的細胞比例，判斷 12/20 週 valve 直徑暴增是否來自細胞主導的 root remodeling。
3. Method:

Ki67 是一個只在細胞「正在分裂」(G1、S、G2、M 期) 時才會出現的細胞核蛋白；細胞如果處於「休息中」的 G0 期，Ki67 就消失。Ki67⁺ 等同「這顆細胞正在或剛剛處於分裂程序中」——活的但不分裂的細胞 (例如成熟內皮、間質細胞) 都不會被染到，死掉的細胞細胞核會崩解、也不會被誤計，所以 Ki67⁺ 比例幾乎直接等於「正在分裂的細胞佔比」，是組織重塑活躍度的乾淨指標。染色用一抗 + 二抗兩階段：一抗 (abcam ab15580，稀釋 1:100) 認 Ki67，本身不發光；二抗認一抗、上面接 Alexa Fluor 647 螢光分子，會發紅色螢光。同時用 Hoechst 33342 把所有細胞核染藍——它是能穿過細胞膜、卡進 DNA 雙螺旋小溝的螢光染料，每個有 DNA 的細胞核都會被染到，跟細胞週期無關。所以紅核是分子 (分裂中)、藍核是分母 (全部細胞)，兩者疊在同一張螢光照片上，數紅核 ÷ 藍核就是 Ki67⁺ 比例。最後在 Leica DMi8 widefield 螢光顯微鏡下拍照計數。

兩階段染色 (indirect immunostaining) 的設計用意是訊號放大：一抗只認目標、本身不發光；二抗認一抗的 Fc 段、上面接螢光分子，而且每根一抗可以被好幾根二抗黏住，等於一個 Ki67 分子最終會被好幾個螢光分子標記，訊號比直接用一個帶螢光的抗體強。代價是會稍微多點背景訊號，所以染色前必須先用 5% normal donkey serum / PBS 做 blocking，把非特異性結合位點封住。如果 blocking 沒做好，抗體會非特異性黏到 ECM、膠原蛋白、結構蛋白上，造成整個切片瀰漫紅光、訊號不集中在細胞核——「分裂 vs 不分裂」的對比被洗淡，計數會大幅高估 Ki67⁺ 比例。標準診斷法是跑「不加一抗」(no primary antibody) 的陰性對照：理論上應該完全暗，如果出現紅訊號就代表 blocking 失敗或二抗有非特異性結合，整批染色都要重做。

瓣膜在解剖上分兩塊：root（外圍那層管壁、像血管壁）與 leaflet（裡面三片可開合的葉片）。兩塊受的力完全不同——root 受徑向擴張、leaflet 在心臟舒張時受壓貼合中央——細胞反應也應該不一樣，所以作者分開數，避免區域訊號被洗掉。比較設計上，Gen 1 四隻羊中，兩隻 (12 週和 20 週) 突然出現 valve 直徑暴增提前犧牲，另兩隻 (36 週和 62 週) 沒有暴增、撐到計畫終點。直徑為什麼突然變大？兩個假說對 Ki67 有完全不同預測。假說 A 是 passive stretch (被動拉鬆)：血壓把 ECM 直接撐大、不需要新細胞，這時候 dilated valve 跟未 dilated valve 的 Ki67 應該差不多。假說 B 是 cell-driven remodeling (細胞主導重塑)：要把 root 撐大又不失去結構完整性，細胞要瘋狂分裂、合成新的 ECM 填進去支撐，這時候 dilated valve 的 Ki67 會明顯升高。實驗結果是 12/20 週 root Ki67⁺ = 1.23 ± 0.59%、36/62 週 root Ki67⁺ = 0.20 ± 0.29%，差約 6 倍，跟假說 B 一致、不支持假說 A。這就是 Ki67 比較對機制歸因的關鍵作用。

Ki67⁺ 比例的可信度仰賴兩個操作標準。第一是抽樣量：組織不均勻、Ki67⁺ 細胞常集中在 remodeling 熱點，附近區域可能幾乎沒分裂細胞，只拍一兩張照片剛好命中熱點會大幅高估、命中冷區會大幅低估。所以本研究每個時間點至少拍 15 個視野、總共超過 6000 個細胞統計 (data file S3)——15 視野把空間異質性平均掉，6000 細胞讓比例只有 1% 的 rare event 仍能拿到統計穩定的分子 (1.23% × 6000 ≈ 74 個 Ki67⁺ 細胞)。第二是切片厚度標準化：切片厚一點 (例如 20 µm) 同一視野細胞數變多但細胞核可能在 z 軸上重疊、被當成兩顆 (double-counting)；切片薄一點 (例如 5 µm) 可能把一顆細胞核切成兩半、每半只看到部分 Ki67 訊號、被誤判為陰性。本研究固定 9 µm 切片厚度，是「單核被切到一次、完整 Ki67 訊號保留」的常用平衡點，讓不同 explant 之間的計數能直接比較。

4. 工具與材料:

   - **Ki67**: 只在細胞分裂期 (G1/S/G2/M) 表現的細胞核蛋白；Ki67⁺ 等於「正在或剛剛分裂」。
   - **abcam ab15580 (1:100)**: 本研究使用的 Ki67 一抗，認 Ki67 蛋白本身不發光。
   - **Alexa Fluor 647 secondary antibody**: 認一抗 Fc 段並接紅色螢光分子，提供訊號放大。
   - **Hoechst 33342**: 細胞核染色劑，卡進 DNA 雙螺旋小溝把所有細胞核染藍，當分母用。
   - **indirect immunostaining (一抗+二抗)**: 兩階段染色策略，訊號放大但會多點背景，需要 blocking。
   - **blocking (5% normal donkey serum)**: 染色前先封住組織非特異性結合位點，避免抗體亂黏造成背景。
   - **Leica DMi8 widefield microscope**: 本研究使用的螢光顯微鏡，搭配 Leica DFC 7000T 相機與 Leica Application Suite X 軟體拍照。
   - **≥ 15 fields / > 6000 cells**: 本研究計數抽樣標準；15 視野平均空間異質性，6000 細胞讓 1% rare event 統計穩定。
   - **Ki67⁺ proportion**: 紅染核 ÷ 藍染核的百分比；12/20 週 root = 1.23 ± 0.59%、36/62 週 root = 0.20 ± 0.29%。
   - **9 µm cryo-section**: 固定的切片厚度，避免細胞核 z 軸重疊或被切半。

5. 與此篇文章的關係:

這篇 paper 的核心宣稱是工程瓣膜能「跟著小羊一起長大」，而 Gen 1 四隻羊中有兩隻在 12/20 週出現 root 直徑暴增、提前犧牲，作者必須回答這個暴增究竟是血壓被動撐開 ECM (passive stretch)、還是細胞主動分裂與重塑 (cell-driven remodeling)——兩個假說對臨床轉譯意義完全相反。Ki67 增殖定量在此扮演機制歸因的關鍵角色：因為 Ki67 只在 G1/S/G2/M 期表達、G0 靜止期消失，Ki67⁺ 比例幾乎等於「正在分裂的細胞佔比」，是判斷組織是否主動 remodel 的乾淨指標。作者把 root 與 leaflet 分開計數，得到 12/20 週 root Ki67⁺ = 1.23 ± 0.59%、36/62 週 root = 0.20 ± 0.29%，約 6 倍差距直接支持細胞主導 remodeling 假說、推翻被動拉鬆假說。這個結果必須搭配 modified Hoechst 細胞密度與 hydroxyproline collagen 量化才能完整詮釋——光看比例不知道絕對細胞數，光看密度不知道是新分裂還是累積結果，三者一致才能宣稱 12/20 週是 root 主動 remodel 的窗口。同時 Ki67 結果也支撐 Gen 2 sleeve 設計的合理性：既然 root 暴增來自細胞主動分裂，物理約束 (sleeve) 才能在不阻斷細胞活動的前提下抑制過度擴張。
