# Fibroblast-fibrin 組織管培養與去細胞化（生物工程組織管製造）

1. 引用自哪篇 paper: pediatric-tri-tube-valved-conduit
2. Outline (任務主線): 在試管內由綿羊真皮 fibroblast 產出環向對齊、機械性質接近原生肺動脈瓣的膠原蛋白管，作為製作 valve 的「原料管」。
3. Method:

作者想要的「原料管」其實是一根用膠原蛋白織成的圓柱布管，作法是讓細胞自己織出來。他們先把綿羊真皮纖維母細胞 (ovine dermal fibroblasts, ODFs) 跟一鍋會凝固的「假血塊」材料拌在一起——也就是兩種血液凝固蛋白 fibrinogen、thrombin 加上鈣離子——混合後幾分鐘內就凝成圓柱形果凍，灌進直徑 16 mm 的玻璃管模具裡。這層假血塊就是「臨時鷹架 (sacrificial scaffold)」，一凝固就把細胞均勻鎖在三維空間裡，等於先給細胞一個立刻可以踩的立體地板；之後細胞會慢慢把它分解掉，在原位織出自己的膠原蛋白頂替。前兩週是「定型期」：作者刻意讓果凍沿著長度方向自由變短，但直徑被裡面的玻璃心軸 (mandrel) 撐著縮不下去；細胞在這種「環向被卡住、軸向能動」的張力場裡會自動把膠原蛋白纖維順著被卡住的環向排列 (circumferentially aligned collagen)，剛好對應原生肺動脈瓣葉片的環向走向。兩週後再把管子鎖死在心軸上，剩下七週讓細胞專心增加膠原蛋白密度、把纖維彼此交聯變硬，總共九週才能熟成一根機械性質接近原生肺動脈瓣的膠原蛋白管。

整段培養液裡，最不能省的兩個成分是 insulin (2 µg/ml) 與 ascorbic acid (50 µg/ml)。Insulin 是讓纖維母細胞保持活躍生產模式的訊號，沒加細胞會懶散、產量驟降。Ascorbic acid（維他命 C）則是膠原蛋白合成路線上的必要輔助分子：細胞剛織出來的膠原蛋白鏈得先被一個叫 prolyl-4-hydroxylase 的酵素在 proline 殘基上掛一個氫氧基 (-OH)，三條鏈才能扣在一起捲成穩定的三股螺旋。沒有維他命 C 這個酵素就停擺，細胞拼命合成的膠原蛋白會在合成路上散掉，整根管子根本撐不起張力。

管子養到第九週時裡面塞滿了當初負責織布的細胞，這時要進入「去細胞化」階段：把細胞洗掉、只留空的膠原蛋白骨架。作者用兩種清潔劑接力清洗——SDS 是「強力清潔劑」，能很快把細胞膜整個拆掉、把蛋白質溶出來，但泡太久會連膠原蛋白網一起變性，所以只泡 6 小時就停；Triton X-100 是「溫和清潔劑」，泡 30 分鐘負責收尾、把殘餘脂質帶走，不會傷到膠原蛋白。最後再用 DNA 分解酶 (DNase, 2 U/ml) 過夜把斷掉的 DNA 碎片消化光。洗完後留下的就是一層空的膠原蛋白管壁，不帶活細胞、沒有可被免疫系統認出的細胞表面標記，植入到別的個體 (allogeneic) 也不會被排斥。

整套流程有兩個容易壞掉的環節值得特別記住。第一是心軸直徑挑錯：心軸太鬆、細胞感受不到「環向被卡住、軸向能縮」的不對等張力，就會把膠原蛋白亂排，管壁變成四面八方一樣硬 (isotropic)。但原生肺動脈瓣葉片本來就是環向比軸向硬 2 倍以上的非等向結構 (anisotropy)，亂排的管子做成瓣膜後，在心臟舒張壓力下會往不該變形的方向被撐開，很快漏血。第二是去細胞化偷工：SDS 泡的時間不夠、或者跳過 DNase，殘留的細胞表面標記與斷裂 DNA 碎片會被宿主免疫系統認成外來威脅，植入後幾週內引發慢性發炎，發炎細胞接著吸引鈣離子沉積，整顆瓣膜走上鈣化變硬的死路——這正是傳統化學固定動物瓣膜的失效模式。

4. 工具與材料:

   - **Ovine dermal fibroblasts (ODFs)**: 綿羊真皮纖維母細胞，本研究的「織布工」，負責在試管內織出膠原蛋白管。
   - **Fibrinogen + Thrombin (fibrin gel)**: 兩種血液凝固蛋白與鈣離子混合後幾分鐘內凝成「假血塊」，當作把細胞均勻鎖進三維空間的臨時鷹架 (sacrificial scaffold)。
   - **Mandrel (16 mm ID 玻璃心軸)**: 圓柱形玻璃模具，撐住果凍的直徑、不讓它環向縮，引導細胞沿環向排列膠原蛋白。
   - **Circumferentially aligned collagen**: 因不對等張力場而產生的膠原蛋白環向走向，模擬原生肺動脈瓣葉片的非等向 (anisotropy) 結構。
   - **Insulin (2 µg/ml)**: 保持纖維母細胞「活躍生產模式」的訊號分子。
   - **Ascorbic acid (50 µg/ml)**: 維他命 C，prolyl-4-hydroxylase 酵素合成穩定膠原蛋白三股螺旋所必需的輔助因子。
   - **SDS (1%, 6 hr)**: 強力離子型清潔劑，第一步快速拆掉細胞膜、溶出蛋白質，泡 6 小時控時以免變性膠原蛋白。
   - **Triton X-100 (1%, 30 min)**: 非離子型溫和清潔劑，第二步收尾、帶走殘餘脂質而不傷膠原蛋白。
   - **DNase (2 U/ml, overnight)**: DNA 分解酶，把殘留 DNA 碎片消化乾淨，避免免疫排斥與下游鈣化。
   - **Decellularized tissue tube**: 九週培養 + 清潔劑去細胞化後得到的空膠原蛋白管，可冷藏保存半年以上備用，作為後續瓣膜組裝的「原料管」。

5. 與此篇文章的關係:

這篇 paper 想解決的核心難題是：兒科病人現有的化學固定生物瓣（Hancock 豬瓣、Contegra 牛瓣）因 glutaraldehyde 交聯而失去生長能力、又容易鈣化，小孩一生因此要動 1–5 次以上開心手術；作者的對策是改用「能跟著小孩一起長大」的工程組織瓣，而本 module 正是整個瓣膜的「原料製造」起點——若沒有這根管，後續所有 tri-tube 縫合、植入、長期追蹤都無從談起。用綿羊真皮 fibroblast 在 fibrin 假血塊裡自己織出環向對齊的膠原蛋白管，最大好處是同時拿到三件難得的特性：機械性質（modulus、UTS、anisotropy）接近原生肺動脈瓣葉、fibrin 鷹架最後會被細胞吃光所以剩下的全是「細胞自產 ECM」可乾淨歸因、再經 SDS/Triton/DNase 去細胞化後沒有可被免疫系統認出的細胞標記，植入異體 (allogeneic) 不排斥、可冷藏半年作為現成貨架材料。此 module 在 method 鏈的最上游：它的輸出（16 mm ID 去細胞化組織管）會被 module B 裁切縫成 tri-tube valve，再經 module E (Bose Durapulse) 與 module F (pulse duplicator) 做體外驗證，最後才進入 module C 的生長羊肺動脈植入；而 module K (ECM proteomics) 也回頭量化此 module 不同 batch 的蛋白組成，用來排除「動物結果差異來自原料 batch 不穩」的替代假說，因此這根管的品質直接決定了整篇研究的成敗。
