# 軸向 / 環向單軸拉伸 (Strain-to-failure) 力學測試

1. 引用自哪篇 paper: pediatric-tri-tube-valved-conduit
2. Outline (任務主線): 量化 tube、explanted valve、native ovine pulmonary leaflet、Contegra leaflet 在環向與軸向的 modulus 與 UTS，驗證工程組織管在植入前後仍維持原生瓣葉等級的張力性能與 anisotropy。
3. Method:

作者把每個組織管、植入後取下的瓣膜、原生羊肺動脈瓣葉與臨床用的 Contegra 牛頸靜脈瓣葉，都切成約 2 mm 寬、10 mm 長的長條，分別沿圓周方向 (環向) 與沿管子長軸方向 (軸向) 各取一條。每條樣本夾進拉力測試機 (Instron tensile testing system) 的客製夾具，先用 0.05 N 這麼小的力把它輕輕拉直、消除鬆弛，再讓夾具持續往外移動、把樣本慢慢拉伸到斷掉。機器同步記錄「拉了多少」與「需要多大的力」，畫成一條應力-應變曲線。應變的算法是「拉到的長度除以原本長度、再取自然對數」($\varepsilon = \ln(L/L_0)$)，應力是「拉力除以原本未變形的橫截面積」(單位 MPa)——這兩個換算等於把不同尺寸的條子全部標準化到同一座標軸，才能跨樣本互相比較。從曲線讀出兩個關鍵指標：剛性指標 (modulus) 是前段近似直線的斜率，斜率越陡代表材料越硬；極限張力 (UTS, ultimate tensile strength) 是整條曲線在斷掉前達到的最高應力，代表組織最多能撐住多大應力才壞掉。

為什麼要分兩個方向各拉一次？因為這種組織不像金屬那樣四面八方都一樣強。組織管在模具內培養時，膠原蛋白纖維幾乎全部排成「沿著圓周一圈一圈」的方向，天生就是「順著圓周比較強、沿著管長比較弱」；原生肺動脈瓣葉也是同樣的方向偏好。這種方向不對等的性質叫 anisotropy。光測一個方向會誤導：可能在環向看起來夠強、軸向卻弱不堪壓。兩個方向都測才能完整描述材料，也才能跟原生瓣葉「對齊比較」。上機前的 0.05 N 預拉直同樣有意義：剛夾上去的組織條多少有點鬆、皺，直接拉的話曲線一開始會有一段「只是在拉直鬆弛」的虛假平坦段，後面的 modulus 與初始長度 $L_0$ 都會算錯。用一個非常小、不會傷材料的力把它輕輕繃直，等於把起跑線對齊到「剛好繃直、還沒受力」這一刻。

光看一個 modulus = 5 MPa 沒辦法判斷好壞——太硬會把血流堵住、太軟撐不住血壓。所以作者同時量了兩個對照：原生羊肺動脈瓣葉 (native ovine pulmonary leaflet) 代表「生理上理想的目標值」，臨床用的 Contegra 牛頸靜脈瓣葉代表「現行療法至少要做到的程度」。工程組織管的 modulus 與 UTS 要落在這兩條參考線附近、特別是貼近原生瓣葉，才能說它在力學上是個合理的替代物。

這套測試的兩個典型失敗模式都會讓結論失真。如果樣本在夾具邊緣先被夾爛、或在還沒真正受力時就從夾具滑開，曲線會「假裝斷掉」——記到的 UTS 比真實值低、應變也因為實際拉伸長度算錯而整段失真；作者用客製夾具就是要讓斷裂發生在中段、而不是夾具邊緣。另一個更隱性的失敗是「只測一個方向」：瓣葉關閉時兩個方向都受力，光測環向會漏掉軸向強度是否足夠，後面用 ANSYS 跑應力模擬時也必須用兩個方向的平均曲線當輸入，少一邊整個模型就沒有依據。

4. 工具與材料:

   - **Uniaxial strain-to-failure tensile test**: 把組織條夾在拉力機兩端、單方向拉到斷掉，記錄整段應力-應變曲線的單軸拉伸測試。
   - **Instron tensile testing system**: 本研究使用的商用拉力測試機，配客製夾具與 0.05 N 預拉直步驟。
   - **Strain (應變)**: 拉到的長度除以原本長度再取自然對數 ($\varepsilon = \ln(L/L_0)$)，把「被拉伸了多少倍」轉成跨樣本可比的小數字。
   - **Stress (應力)**: 拉力除以樣本未變形時的橫截面積，單位 MPa，等於「每單位面積要承受多少力」。
   - **Modulus**: 應力-應變曲線前段直線的斜率，斜率越陡代表組織越硬、想多拉一點要花更多力。
   - **UTS (Ultimate Tensile Strength)**: 曲線斷掉前達到的最高應力值，代表組織最多能撐住多大應力才壞。
   - **Circumferential / Axial 取樣**: 沿圓周方向 (環向) 與沿管子長軸方向 (軸向) 各取一條長條，量出材料的方向性。
   - **Anisotropy**: 同一塊組織在不同方向強度不同的性質；本組織管環向遠強於軸向，與原生瓣葉一致。
   - **0.05 N 預拉直**: 上機前用很小的力把樣本繃直、消除鬆弛，讓起跑線對齊「剛好繃直但還沒受力」這一刻，避免曲線初段失真。
   - **Native ovine pulmonary leaflet 對照**: 原生羊肺動脈瓣葉，作為「生理上理想的目標值」的參考線。
   - **Contegra leaflet 對照**: 臨床標準的牛頸靜脈瓣葉，作為「現行療法至少要做到的程度」的參考線。

5. 與此篇文章的關係:

這篇 paper 想證明由 fibroblast 培養出來的工程組織管所縫成的 tri-tube 瓣膜，植入生長中的羊體內後仍能維持原生肺動脈瓣等級的力學表現，而單軸拉伸測試正是把「硬度 (modulus) 與斷裂強度 (UTS)」這兩個力學底線量化出來的核心手段。作者把 preimplant tube、explanted valve、原生羊肺動脈瓣葉、Contegra 牛頸靜脈瓣葉一起送進同一台 Instron，分別沿環向與軸向各拉一條，好處是能在同一條起跑線上比較「金標準 (native leaflet)」「臨床對手 (Contegra)」與工程組織管的差距，並同時驗證組織是否複製了原生瓣葉「環向硬、軸向稍軟」的 anisotropy——這是瓣葉能正常閉合的功能基礎。在整體研究計畫裡，這套測試與生化定量 (collagen / DNA / Ki67) 並列檢查植入後組織的「結構是否仍夠強、細胞是否真的長進去」，用來區辨 valve 在 in vivo 變大究竟是 somatic growth 還是 aneurysmal stretching；同時，它測出的軸向與環向應力-應變平均曲線又被反過來餵進 ANSYS Mooney-Rivlin 有限元素模型，當作 tube-in-tube 與 tri-tube 應力分佈比較的本構輸入，少了這套力學數據，ANSYS 的設計理由與植入後的生長宣稱都會失去依據。
