# 結構參數量化（膜厚、APL、密度剖面）

1. 引用自哪篇 paper: solid-ordered-bilayer-mechanics
2. Outline (任務主線): 結構參數量化（膜厚、APL、密度剖面）
3. Method:
   結構參數的第一個產物是膜厚 (membrane thickness, MT)。MD 裡每顆脂質都帶一個磷原子在頭基位置，作者把它當作這片膜的「位置代理人」。每一幀都對三組磷原子分別做球面擬合：內側葉片那一群磷原子擬一顆球得到 r_inner、外側那一群擬一顆得到 r_outer、把全部磷原子合起來擬一顆得到 r_vesicle。膜厚 MT 就是 r_outer 減 r_inner——也就是兩層磷原子球面的厚度差。挑磷原子當代理人有兩個理由：磷原子位在頭基與水接觸的界面上，垂直方向分布特別緊，是整顆脂質定位最穩的座標；而且實驗用 X-ray 散射或 NMR 量膜厚時量的其實就是兩層磷酸根面之間的距離，這樣 MD 數字才能直接跟實驗（POPC 的 3.05 nm NMR 數字、3.9 nm X-ray 數字）對上。整段 MT 是用最後 12–65 ns 取時間平均算出來，不是用單幀——因為單幀會把瞬時皺褶當成真的形變，誤差量級就有 1 Å，但固態系統之間真正想分辨的 MT 差異常常落在 0.05 Å 級，不取時間平均就會被瞬時雜訊整個淹沒。

   第二個產物是每脂面積 (area per lipid, APL)：把一片葉片的表面積平均分給每顆脂質、每顆可以分到多少地。平面膜很簡單，但囊泡是球面，外葉片那層球一定比內葉片那層球面積大，且兩邊脂質數本來就不同。所以作者依 Braun & Sachs (ref 26) 的公式分開算：$\mathrm{APL}_{\mathrm{inner}} = 4\pi r_{\mathrm{inner}}^{2} / n_{L,\mathrm{inner}}$（方程式 16）、$\mathrm{APL}_{\mathrm{outer}} = 4\pi r_{\mathrm{outer}}^{2} / n_{L,\mathrm{outer}}$（方程式 17），再用 $\mathrm{APL}_{\mathrm{vesicle}} = 4\pi r_{\mathrm{vesicle}}^{2} / [(n_{L,\mathrm{inner}} + n_{L,\mathrm{outer}})/2]$（方程式 15）取整顆平均。建構系統時內外葉片 APL 起始值已先做了 0.95 / 1.05 曲率修正（子項 C），那是起始給足、避免一開始就出現脂質太鬆或太擠的人為缺陷。但模擬一旦跑起來，水會慢慢穿過膜、雙層自己重新分配，最後 APL 是自組裝平衡的結果。所以分析端必須重新分開量 APL_inner 與 APL_outer 來驗收兩邊有沒有達到合理的內外不對稱。DMPC 系統就在這一步被抓出異常——內葉片 APL（72.1 Å²）比外葉片（56.3 Å²）大很多，背後對應的是模擬中曾自發開出一個短暫的水孔讓內外水密度重新平衡。沒有這層後分析，這種結構訊號會被整顆囊泡的平均 APL 抹掉。

   第三個產物是密度剖面 (density profile)：從球心往外走一條徑向線，會依序穿過內葉片的酯鏈→甘油酯→頭基→水層→頭基→甘油酯→外葉片的酯鏈。作者把每一顆原子按它在脂質分子上的位置分成三類——頭基 (headgroups)、甘油酯 (carbonyl–glycerol)、酯鏈 (acyl chains)。每一幀算原子到囊泡質心的距離，至少累積最後 100 幀做直方圖，就得到不同部位的原子各自分布在哪個半徑這張地圖。每個尖峰再用常態分布 (normal distribution) 擬合，同時拿到尖峰位置（這段在膜厚裡的座標）與尖峰寬度（這段被熱漲落晃得有多廣）。常態分布在這裡夠用是因為單一一段在熱平衡下會做許多小幅、彼此獨立的位移，按中央極限定理就會收斂到鐘形；對固態膜仍適用——尾巴雖然站得直、移動受限，但仍在自己的格點附近小幅振動，尖峰只是變得比液態膜窄。如果只取 10 幀，直方圖會充滿尖刺、出現假副峰，擬合出來的均值與寬度都不可靠；100 幀才能把熱平衡的統計收滿、把 DMPC 酯鏈尖峰比 DSPC 窄、HSPC 混合膜的尖峰會不會變寬這類細微差異浮出來。

   MT、APL、密度剖面這三組結構數字不是只給結構表填空：它們是後續力學參數（彎曲剛性 κ 與面積壓縮率 KA）的物理脈絡。MT 提供 Helfrich 模型的厚度尺度、APL 是脂質互相卡得多緊的最直接量度、密度剖面則告訴你尾巴有沒有站直凍住——三者一起決定了固態膜為什麼比液態膜難拉開、又為什麼整顆囊泡卻反而好彎。這也是本子項放在力學參數萃取之前的原因：先把結構訊號量穩，後面的反直覺力學結果才能有立足點。
4. 工具與材料:
   - **membrane thickness (MT)**: 雙層的厚度，定義為內外葉片磷原子球面半徑之差 r_outer − r_inner；對應實驗 X-ray / NMR 量到的磷－磷距離。
   - **area per lipid (APL)**: 每脂面積；囊泡分內葉片 APL_inner、外葉片 APL_outer、整顆 APL_vesicle 三個版本（方程式 15–17）。
   - **phosphorus atom (leaflet marker)**: 脂質頭基的磷原子，因垂直位置分布緊密、又能對應實驗，被用作 MD 中葉片位置的代理人。
   - **r_inner / r_outer / r_vesicle**: 對每幀內側、外側、全部磷原子分別做球面擬合得到的三顆半徑；提供 MT 與三個 APL 公式的幾何輸入。
   - **density profile**: 把每顆原子到囊泡質心的距離累積成徑向直方圖，呈現「頭基／甘油酯／酯鏈」三段在膜厚裡各自落在哪。
   - **headgroups / carbonyl–glycerol / acyl chains**: 把脂質分子拆成三個結構區的標準分段，分別管水合界面、酯鍵連接、疏水核心。
   - **normal distribution fit**: 對密度剖面每個尖峰做常態分布擬合，得到尖峰位置與寬度兩個摘要數字以便跨系統比較。
   - **0.95 / 1.05 curvature correction**: 建構階段對內外葉片 APL 各別乘上 0.95 與 1.05 的初始修正（Braun & Sachs, ref 26）；本子項負責在平衡後重新分計 APL 驗收結果。
5. 與此篇文章的關係:
   在《Mechanical Properties Determination of DMPC, DPPC, DSPC, and HSPC Solid-Ordered Bilayers》這篇文章中，作者要解釋固態膜囊泡為何反而比液態好彎。他們用同一份全囊泡 MD 軌跡量化膜厚 MT、每葉片 APL 與三段密度剖面，作為彎曲剛性 κ 與面積壓縮率 KA 的物理脈絡。這套結構量化吃進 NAMD/CHARMM36 軌跡的磷原子座標，輸出 Table 1 結構欄與 Figure 1B、Figure 2 的密度剖面，並抓出 DMPC 系統自發水孔等異常，為下游力學詮釋奠基。
