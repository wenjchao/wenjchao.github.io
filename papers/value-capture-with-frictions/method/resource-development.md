# 資源開發擴充 (Resource-development extension)

1. 引用自哪篇 paper: value-capture-with-frictions
2. Outline (任務主線): 資源開發擴充 (Resource-development extension)
3. Method:
   這個模塊把『開戰前先練功』寫進 biform game 的第一幕。每家供應商選一個非負的資源投資 r_i，把自己帶給買方的價值從 V_i 升級成 V_i + r_i，成本是 c(r_i) = c r_i^2 二次成本（練越多越貴，c 越大代表這個產業練功越貴）。升級後價值決定誰領先誰落後，Stage 2 的期望利潤分成兩段（paper Eq. 49）：誰的 V_i + r_i 較大就拿『雙寡頭領先項』(1−f)(1−α)((V_i+r_i)−(1−f)(V_j+r_j)) 扣掉自己的 c r_i^2；落後者拿『獨佔窗口項』f(1−f)(1−α)(V_i+r_i) 扣掉自己的 c r_i^2。Lemma 1 證明邊際收益只能是兩個值之一——預期領先時 (1−α)(1−f)、預期落後時 (1−α)f(1−f)；對二次成本取一階條件得 r^H = (1−α)(1−f) / 2c、r^L = (1−α)f(1−f) / 2c = f · r^H，差距 r^H − r^L = (1−α)(1−f)^2 / 2c 隨摩擦上升而下降。Proposition 7 給出兩種非對稱均衡：sustaining (r^H, r^L) 永遠存在；leapfrogging (r^L, r^H) 只在三條件全部成立時才存在，最緊的是 V_1 − V_2 < (1−α)(1−f)^2 / 4c（paper Inequality 1）。
   為什麼領先的邊際收益 (1−α)(1−f) 比落後的 (1−α)f(1−f) 大這麼多？領先者多投一塊錢能在『買方看到自己』這個機率 1−f 的所有情境兌現——不管對手有沒有被察覺都行；落後者多投一塊錢只能在『強對手剛好沒被察覺（機率 f）且自己被察覺（機率 1−f）』的聯合機率 f(1−f) 窄窗裡兌現。比值就是 f：摩擦越大窄窗越寬，落後者投資的邊際回報相對拉近，但只要 f < 1 領先者永遠在練功回報率上佔上風——這正是模型內生出現強弱分化的根本動力。也正因如此，分段利潤函數會逼出『對稱均衡不可能』：兩家都重押 (r^H, r^H) 時弱者降到 r^L 能省成本而損失極小（Π_2(r^H, r^L) − Π_2(r^H, r^H) = (1/4c)(1−α)^2(1−f)^4 > 0），弱者必偏離；對 (r^L, r^L) 同樣有強者必偏離。對稱解被分段結構強制排除，留下的只能是 (r^H, r^L) 或 (r^L, r^H) 兩種非對稱解。
   為什麼挑二次成本 c r_i^2 而非線性成本？二次成本的邊際成本 2c r_i 隨投資線性遞增，給定一個邊際收益就能用 FOC 解出唯一的最佳 r。換成線性成本邊際成本是常數，最佳投資會跳到 0 或無窮大，無法形成『r^H、r^L 兩個有限水準』的乾淨刻畫——Proposition 7 與 Inequality 1 那套閉式都依賴二次成本提供的內部最佳解。另一個關鍵設計是用 Schelling focal point 選 leapfrogging vs sustaining：兩個均衡同時成立時模型自己選不出誰勝出（兩者都站得住腳），作者引用 Schelling (1960) 的概念——哪一個成真，靠雙方對『誰會是領導者』形成的共同預期，誰先讓對方相信自己會贏，那個劇本就自我實現。這個設計把『管理者塑造市場預期』連結到內生競爭優勢，是論文管理學涵義的核心錨點。
   兩個地方一失手 Proposition 7 的結構會崩。第一，只檢查一個條件就宣告 leapfrogging：(r^L, r^H) 要存在必須三條件同時成立——(a) 可行性 V_1 + r^L < V_2 + r^H、(b) 弱供應商不偏離、(c) 強供應商不偏離；最緊的是 (a)，化簡成 V_1 − V_2 < (1−α)(1−f)^2 / 4c（paper Inequality 1）。只查一個就宣告，可能漏報另兩個條件失效的情境，把不可逆轉的市場誤判成可逆轉。第二，把分段利潤函數當連續函數最佳化：函數在 V_i + r_i = V_j + r_j 那條邊界上有 kink（斜率不連續），代表領先/落後身份切換；忽略 kink 會抹掉『誰是領先者』的離散切換，兩個離散水準的差距、leapfrogging 的存在條件、對稱均衡不存在性都會消失。所以必須把利潤拆成兩段分別求 FOC、再用差分檢驗哪個是 Nash。
4. 工具與材料:
   - **資源投資 r_i ≥ 0**: Stage 1 供應商的新增決策變數，把自己帶給買方的價值從 V_i 升級到 V_i + r_i。
   - **二次成本 c r_i^2**: 凸成本函數，c > 0 是產業內練功難度的單一刻畫；提供內部 FOC 解 r^H、r^L 的關鍵。
   - **兩個最佳投資水準 r^H, r^L**: r^H = (1−α)(1−f) / 2c（預期領先）、r^L = f · r^H（預期落後），來自 Lemma 1。
   - **分段利潤函數 (piecewise profit, paper Eq. 49)**: 領先/落後身份切換造成的 kink，是兩水準均衡與 leapfrogging 多重均衡的數理根源。
   - **sustaining vs leapfrogging 均衡**: (r^H, r^L) 強者守位永遠存在；(r^L, r^H) 弱者翻盤只在 V_1 − V_2 < (1−α)(1−f)^2 / 4c 等三條件下存在。
   - **Schelling focal point**: 多重均衡選擇的概念裝置，把『誰會領先』交給雙方共同預期，連結到管理者塑造市場預期的策略意涵。
5. 與此篇文章的關係:
   在《Value Creation and Value Capture with Frictions》這篇文章中，作者為了把資源基礎觀（RBV）的『firm-level 異質性是否內生出現、初始優勢是否可持續』正式化，採用了在 biform 第一幕加入二次成本資源投資的擴充。它解決了 RBV 傳統文獻只能口頭討論可持續優勢、缺乏可比較靜態形式模型的瓶頸，並把 § 2B 摩擦機率與 § 2A 特徵函數產出的雙段 Π_i 當作 Stage 1 投資 FOC 的輸入，產出 Proposition 7 與 Inequality 1 描述的 sustaining / leapfrogging 多重均衡判準。
