# 進入成本擴充 (Entry-barrier extension)

1. 引用自哪篇 paper: value-capture-with-frictions
2. Outline (任務主線): 進入成本擴充 (Entry-barrier extension)
3. Method:
   作者在 biform 的 Stage 1 加上一個二元選擇：要不要花一筆固定成本 F 進場？進場了下半場才能去分餅；不進場淨利潤直接是 0 ($\Pi_i^{NE} = 0$)。下半場依照進場的廠商組合分三種情境：獨佔時拿到 $\Pi_i^M = (1-f)(1-\alpha)V_i$；雙寡頭下強者拿到 $\Pi_1^D = (1-f)(1-\alpha)(V_1 - (1-f)V_2)$；雙寡頭下弱者拿到 $\Pi_2^D = (1-f) \cdot f \cdot (1-\alpha)V_2$ ——亦即只有在『自己今天現身、強者卻沒被買方看見』那一格才賺得到錢。Stage 1 就是 2×2 賽局，每家問『我這個策略下扣掉 F 能不能不虧』；例如『1 進、2 不進』要成立必須同時滿足 $\Pi_1^M \geq F$ 且 $\Pi_2^D \leq F$。整套用閉式不等式判斷，不需要動 fixed-point 定理。
   為什麼這個擴充能讓 rivalry 與進入威脅耦合在同一個 f 軸上？關鍵在於進場的『回本條件』 Π_i(f) ≥ F 的左邊是 f 的函數，右邊是常數。當 f 滑動時，左邊那條曲線（倒 U 型）會多次穿越 F 這條水平線——每穿越一次某家的進場決策就翻面，整個市場結構（壟斷 / 雙寡頭 / 無進入）跟著相變。為了讓相圖保留『雙廠進入』這個有意義的中段，作者還加上 F ≤ ¼(1−α)V2 這個可行性條件——這正是弱者進場收益 $(1-f) \cdot f \cdot (1-\alpha)V_2$ 在 f=1/2 達到的最大值。如果 F 超過這個峰值，弱者永遠回不了本，相圖就只剩『獨佔 / 無進入』兩段，論文最有意思的不連續跳躍結論就失去中段（footnote 15 推導）。
   如果把 rivalry 與 entry 分開兩個模型分析會錯在哪？教科書直覺 (Porter 五力 / Grant 2005) 會把『同業競爭強度』與『新進入威脅』當兩條獨立力量分開檢查，先評估 rivalry 高不高，再評估進入門檻夠不夠擋，最後加總得到行業吸引力。但這個擴充顯示這樣加總會錯——f 微調時 $\Pi_2^D(f)$ 滑過 F 會讓弱者突然進場或突然退場，整個行業利潤在 f 軸上呈不連續跳躍。分開分析會看到 rivalry 增加時利潤『平滑下降』；嵌在同一 biform 才能看到『rivalry 增加 → 弱者退場 → 強者獨佔 → 行業利潤瞬間跳升』這種反直覺結果。這直接挑戰 Porter 五力可獨立分析的教科書神話。
4. 工具與材料:
   - **固定進入成本 F**: Stage 1 進場要繳的一筆固定費用，是 entry barrier 的單參數表示。
   - **Enter / Stay out**: Stage 1 的二元選擇行動空間。
   - **Π_i^M = (1−f)(1−α)V_i**: 獨佔下廠商 i 的 Stage 2 value capture 公式。
   - **Π_1^D = (1−f)(1−α)(V_1−(1−f)V_2)**: 雙寡頭下強者的 value capture，反映強者只能多分到比弱者多創造的那塊（再乘上弱者出現機率）。
   - **Π_2^D = (1−f)·f·(1−α)V_2**: 雙寡頭下弱者的 value capture——只有在強者今天沒被買方看見的那一格才賺到錢。
   - **可行性條件 F ≤ ¼(1−α)V2**: 保留雙廠進入區間非空的必要前提；對應弱者進場收益倒 U 型在 f=1/2 的峰值。
   - **pure strategy Nash equilibrium**: Stage 1 在 {Enter, Stay out} 二元行動空間上的均衡，用閉式不等式判斷。
5. 與此篇文章的關係:
   在《Value Creation and Value Capture with Frictions》這篇文章中，作者為了挑戰 Porter 五力『可獨立分析』的教科書直覺，採用了 entry-barrier extension。這解決的瓶頸是：若把 rivalry 與 entry threat 拆成兩個獨立力量加總，會錯失『摩擦 f 微調讓弱者突然進場或退場』造成的不連續行業利潤跳躍。這個擴充吃 2-A 的特徵函數、2-B 的摩擦機率、2-C 的 biform 時序為輸入，產出顯式的三段 value-capture 公式 (Eqs. 37-39) 與閉式 Nash 不等式，直接交給後續『進入均衡 case 分析』把 f 軸切成四段相圖。
